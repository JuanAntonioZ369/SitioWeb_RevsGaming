01

Introduction

RevsGaming is a retro game launcher for Windows. It automatically organizes your collection, launches the correct emulators without any configuration, and connects you with friends for online play.

It supports PS1 via Mednafen, and via RetroArch: NES, SNES, GBA, Game Boy, N64, Mega Drive, Arcade, Saturn and PS2. Emulators are bundled in the installer — no separate installation needed.

Current version: v0.84.0 Beta — actively in development. Some features may change in future versions.
02

Installation

Windows 10 / 11 (64-bit)
1
Download the installer Get the .exe file from the releases page on GitHub.
2
Run the installer Choose an installation folder. Mednafen, RetroArch and all dependencies are installed automatically.
3
Open RevsGaming Use the desktop shortcut created by the installer.
4
Create an account or play as guest You can register to access cloud saves and the friends system, or enter as a guest with a nickname.
5
PS1 BIOS auto-download If the app detects missing PS1 BIOS files, it downloads them automatically on first use.
03

Organizing your games

The default folder is games/ inside the installation directory. You can change it at any time from the Games Folder button in the app sidebar.

Recommended structure

Folder structure 
games/
├── PS1/
│   └── Crash Bandicoot/
│       ├── crash.cue
│       └── crash.bin
└── NES/
    └── Super Mario/
        └── mario.nes
  • Each game should be in its own subfolder.
  • For PS1, the main file is the .cue (which points to the .bin).
  • For other systems: .nes, .sfc, .gba, .n64, .md, etc.
  • The app supports up to 3 levels of subfolders.
04

Playing solo

1
Open your library RevsGaming detects and displays all your games on startup.
2
Select a game Click it, or press A on a connected controller.
3
Choose Solo mode Select Solo on the mode selection screen that appears.
4
The game launches RevsGaming starts the correct emulator automatically. If the RetroArch core is not downloaded yet, it will download it at this point.
PS1 uses Mednafen. NES, SNES, GBA, N64 and other systems use RetroArch with the appropriate core.
05

Play online — 2 players (PS1)

PS1 netplay supports 2 simultaneous players using Mednafen.

As the host

1
Create the room Select a PS1 game → Online → Create Room → choose 2 Players.
2
Get the code A 6-letter code appears (e.g. XKMT4R). Share it with your friend however you like.
3
The game starts automatically Once your friend enters the code and connects, the game launches on both PCs.

As a guest player

1
Go to Join On the online mode screen, select Join.
2
Enter the code Type the 6-letter code the host gave you. The game starts automatically.
Important: Both players must have exactly the same ROM — the same .cue and .bin files. If the files differ, synchronization will fail.
06

Play online — up to 4 players

Available for NES, SNES, GBA, Mega Drive and other systems via RetroArch. The MITM server is located in Lima, Peru, optimized for Latin America.

As the host

1
Create the room Select a game → Online → Create Room → choose 4 Players.
2
Share the numeric code A numeric code appears. Share it with up to 3 friends.
3
Wait for everyone to join The game starts when all players connect or when you decide to begin.

As a guest player

1
Go to Join Select Join on the online mode screen.
2
Enter the code Type the numeric code. The game starts automatically.
07

Friends system

Requires a registered account. The friends system is not available in guest mode.
  • Search for a friend by their username and send a friend request.
  • See in real time who is online and what game they are playing.
  • From your friends list you can invite them directly to your online room.
  • You receive a notification when a friend joins the app.
08

Cloud Saves

Requires a registered account. Cloud saves are not available in guest mode.
  • Saves are automatically synced when you close a game.
  • When launching a game, the most recent saves are downloaded from the cloud.
  • No manual action needed — everything happens in the background.

Supported formats

  • .mcr, .mcs — Mednafen saves (PS1)
  • .srm, .sav, .state* — RetroArch saves and states

Saves are stored in Supabase Storage, encrypted and linked to your user account.

09

Controller input

You can navigate the entire RevsGaming interface with an Xbox or PlayStation controller connected via USB or Bluetooth.

Button App action
D-Pad / Left stick Navigate library and menus
A (Cross) Confirm / Open game
B (Circle) Back / Cancel
Y (Triangle) Toggle list / grid view
Start Open controller detector

Once inside a game, controls depend on the emulator (Mednafen or RetroArch) and its standard button mapping.

10

Frequently asked questions

Where do I put the PS1 BIOS files?
You don't need to. RevsGaming downloads the PS1 BIOS files automatically on first use. If you already have them, the app will detect them.
What if the RetroArch core is missing?
The app downloads it automatically from buildbot.libretro.com the first time you open a game that requires it. Just make sure you have an internet connection.
Are my saves safe?
Yes. Cloud saves are stored in Supabase Storage, encrypted and linked to your user account. Only you have access to them.
Can I change the games folder?
Yes, from the "Games Folder" button in the app sidebar. You can point it to any drive or folder on your PC.
Does guest mode save my progress?
No. Cloud saves require a registered account. In guest mode, saves are stored locally for the duration of the session.
What happens if the beta closes?
You will receive an advance notification inside the app. Your cloud saves will remain safe.

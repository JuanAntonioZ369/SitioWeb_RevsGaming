v0.84.0 June 2026 CURRENT
  • NEW Friends system with real-time presence (Supabase Realtime)
  • NEW Automatic cloud saves via Supabase Storage (.mcr, .mcs, .srm, .sav, .state*)
  • NEW Automatic PS1 BIOS download from a configurable URL in Supabase
  • NEW RetroArch core download from official buildbot with automatic ZIP extraction

  • IMPROVEMENT isTrustedSender() validation on sensitive IPC handlers

  • NEW Login and registration with email/password via Supabase Auth
  • NEW Auto-restore of nickname when logging in from a new PC
  • NEW Configurable beta gate (beta_open flag in app_config)

  • NEW List and grid view with real-time search
  • NEW Manual .cue selector for games outside the main folder

01

Datos que recopilamos

Cuando usas RevsGaming, podemos recopilar los siguientes tipos de información:

  • Dirección de correo electrónico: proporcionada al crear una cuenta. Se usa exclusivamente para autenticación via Supabase Auth.
  • Nickname / apodo: el nombre de usuario que eliges al registrarte o en cualquier momento posterior. Es visible para otros usuarios en el sistema de amigos.
  • Dirección IP: registrada de forma transitoria durante las conexiones de netplay para establecer sesiones entre jugadores a través del servidor MITM en Lima, Perú. No la almacenamos de forma permanente.
  • Saves y estados de juego: archivos de guardado (.mcr, .mcs, .srm, .sav, .state*) que sincronizas voluntariamente mediante la función Cloud Saves. Se almacenan en Supabase Storage bajo tu cuenta.
  • Datos de sesión: token de autenticación generado por Supabase Auth, almacenado localmente en tu dispositivo para mantener la sesión activa entre reinicios de la app.
  • Estado de presencia: cuando el sistema de amigos está activo, se comparte en tiempo real si estás conectado o en qué juego te encuentras. Esto se transmite via Supabase Realtime y no se almacena de forma permanente.
02

Para qué usamos tus datos

Usamos la información recopilada únicamente para las siguientes finalidades:

  • Autenticación: verificar tu identidad al iniciar sesión y proteger el acceso a tu cuenta.
  • Sincronización de saves: subir y restaurar tus archivos de guardado entre distintos dispositivos o tras reinstalar la app.
  • Sistema de amigos: mostrar tu nickname y estado de presencia a los usuarios que hayas agregado como amigos.
  • Conexión netplay: usar tu IP de forma transitoria para establecer la conexión entre los jugadores de una sala. El emulador gestiona esta conexión directamente.
  • Control de acceso a la beta: verificar si la beta está abierta (campo beta_open en Supabase) y notificarte si hay una versión más reciente disponible.
No vendemos, cedemos ni compartimos tus datos personales con terceros con fines publicitarios o comerciales.
03

Servicios de terceros

RevsGaming utiliza los siguientes servicios de terceros para operar:

  • Supabase — plataforma de base de datos, autenticación y almacenamiento. Tu email, nickname y archivos de save se almacenan en la infraestructura de Supabase. Supabase procesa estos datos de acuerdo con su propia política de privacidad: supabase.com/privacy.
Supabase almacena datos en servidores de AWS. Al usar RevsGaming aceptas también las condiciones de Supabase en lo relativo al tratamiento de tus datos.

No integramos servicios de analítica de terceros (Google Analytics, Mixpanel, etc.) ni redes publicitarias.

04

Datos que NO recopilamos

  • Datos de pago: RevsGaming puede ofrecer en el futuro funciones de pago procesadas por Culqi. En ese caso, los datos de tarjeta o método de pago son procesados directamente por Culqi y nunca pasan por nuestros servidores ni los almacenamos.
  • Historial de ROMs o juegos: no registramos qué juegos lanzas, cuánto tiempo juegas ni ninguna estadística de uso de tu biblioteca.
  • Contenido de los archivos de juego: los archivos .cue, .bin u otros formatos de ROM que añades a tu biblioteca no son inspeccionados, analizados ni enviados a ningún servidor.
  • Datos biométricos: no recopilamos ningún tipo de dato biométrico.
05

Retención de datos

Tus datos personales se conservan mientras tu cuenta esté activa en RevsGaming. Esto incluye tu email, nickname y los archivos de save almacenados en Supabase Storage.

Si deseas eliminar tu cuenta y todos los datos asociados, puedes solicitarlo abriendo un Issue en GitHub:

github.com/JuanAntonioZ369/RetroGamingRevs_2.0/issues
Indica tu nickname y el email de tu cuenta. Procesaremos la solicitud en un plazo razonable.

Los datos de presencia en tiempo real (estado online/offline) no se almacenan de forma permanente y se eliminan automáticamente al cerrar sesión.

06

Seguridad

Implementamos medidas técnicas para proteger tus datos:

  • Content Security Policy (CSP): aplicada via onHeadersReceived en la app Electron para prevenir inyecciones de código.
  • Validación de rutas: prevención de path traversal en el acceso a archivos de juego, evitando que código malicioso acceda a archivos fuera de las carpetas permitidas.
  • Validación IPC: los manejadores de mensajes entre procesos verifican isTrustedSender() para aceptar solo comunicaciones legítimas.
  • Comunicación cifrada: toda comunicación con Supabase se realiza sobre HTTPS/TLS.
  • Tokens locales: el token de sesión se almacena localmente en tu dispositivo y no se comparte con terceros.
RevsGaming es software en fase beta. Aunque tomamos medidas de seguridad razonables, no podemos garantizar seguridad absoluta. Si descubres una vulnerabilidad, por favor repórtala via GitHub Issues.
07

Modo invitado

RevsGaming ofrece un modo invitado que permite usar la app sin crear una cuenta. En este modo:

  • No se recopila ni almacena ningún dato personal de forma permanente.
  • Se usa un apodo temporal elegido por el usuario, que no se guarda en ningún servidor.
  • Las funciones de Cloud Saves y sistema de amigos no están disponibles.
  • Al cerrar la app, el apodo temporal se descarta.
08

Tus derechos

En relación a tus datos personales tienes derecho a:

  • Acceso: solicitar información sobre qué datos tenemos almacenados sobre ti.
  • Rectificación: corregir tu nickname o email desde los ajustes de la app.
  • Eliminación: solicitar la eliminación de tu cuenta y todos los datos asociados via GitHub Issues.
  • Portabilidad: tus saves se almacenan en formatos estándar (.mcr, .srm, .sav, .state) que puedes descargar y usar con cualquier emulador compatible.
